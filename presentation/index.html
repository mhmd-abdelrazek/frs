02 — Architecture

High-level Overview

Three core components communicate to capture, store, and display attendance data.

🖥️
ESP8266
Microcontroller + Fingerprint sensor. Reads biometric data on-site.
Wi-Fi / HTTP
☁️
Firestore
Cloud database by Firebase. Stores users, sessions & records remotely.
REST / SDK
📱
Mobile App
Flutter app for managing sessions and viewing attendance.

06 — Hardware

ESP8266 Stack

Embedded components at the physical attendance terminal.

ESP8266 Microcontroller
Wi-Fi microchip with full TCP/IP stack and microcontroller capability. Handles sensor communication, local logic, and cloud sync over HTTP/HTTPS to Firebase.
Tensilica L106 · 80 MHz
Wi-Fi 802.11 b/g/n
UART · GPIO
R307 Fingerprint Module
Optical fingerprint sensor. Captures, enrolls, and verifies fingerprints on-device. Returns a numeric ID that maps to the Firestore users collection.
UART Interface
